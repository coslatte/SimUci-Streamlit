# SimUCI Application Information

## Overview

SimUCI is an ICU (Intensive Care Unit) patient data simulation application, built with Streamlit and SimPy.

## Main Features

- **Patient Simulation**: Discrete event modeling for mechanically ventilated patients
- **Statistical Analysis**: Friedman and Wilcoxon tests for group comparison
- **Data Visualization**: Interactive charts and result tables
- **Prediction**: Machine learning model for outcome prediction

## Architecture

- **Frontend**: Streamlit for user interface
- **Backend**: SimPy for discrete event simulations
- **Data**: Pandas DataFrames for data manipulation
- **Statistics**: SciPy for statistical analysis

## Usage

1. Load patient data from CSV files
2. Configure simulation parameters
3. Run simulations
4. Analyze results and statistics

## System Requirements

- Python 3.9+
- Libraries: streamlit, simpy, pandas, numpy, scikit-learn, scipy
