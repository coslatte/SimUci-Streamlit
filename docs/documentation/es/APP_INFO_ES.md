# Información de la Aplicación SimUCI

## Descripción General

SimUCI es una aplicación de simulación para datos de pacientes en UCI (Unidad de Cuidados Intensivos), construida con Streamlit y SimPy.

## Características Principales

- **Simulación de Pacientes**: Modelado discreto de eventos para pacientes ventilados mecánicamente
- **Análisis Estadístico**: Pruebas de Friedman y Wilcoxon para comparación de grupos
- **Visualización de Datos**: Gráficos interactivos y tablas de resultados
- **Predicción**: Modelo de machine learning para predicción de resultados

## Arquitectura

- **Frontend**: Streamlit para la interfaz de usuario
- **Backend**: SimPy para simulaciones discretas
- **Datos**: Pandas DataFrames para manipulación de datos
- **Estadísticas**: SciPy para análisis estadístico

## Uso

1. Cargar datos de pacientes desde CSV
2. Configurar parámetros de simulación
3. Ejecutar simulaciones
4. Analizar resultados y estadísticas

## Requisitos del Sistema

- Python 3.9+
- Bibliotecas: streamlit, simpy, pandas, numpy, scikit-learn, scipy
