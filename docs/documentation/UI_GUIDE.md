# SimUCI - User Interface Guide

## Overview

SimUCI uses Streamlit to provide an interactive web-based interface.

## Application Structure

### Sidebar
Contains global controls and information:
*   **App Info**: Brief description.
*   **Global Seed**: Control randomness for reproducibility.
*   **Theme**: Toggle light/dark mode.
*   **Version**: Current app version.

### Main Area
Divided into tabs for different workflows:
1.  **Simulation**: Configure patient parameters and run individual simulations.
2.  **Validations**: Compare simulation results against known datasets.
3.  **Comparisons**: Advanced statistical comparison between scenarios.
