# SimUciCausal

Módulo de inferencia causal integrado en SimUCI.

## Objetivo

Estimar el efecto causal de intervenciones clínicas sobre variables
de interés mediante modelos contrafactuales.

## Tecnologías

- DoWhy
- EconML
- Scikit-learn
- SimPy

## Métodos implementados

- Propensity Score Matching
- Double Machine Learning
- Orthogonal Random Forest
- Generalized Random Forest

## Flujo de trabajo

1. Selección de tratamiento
2. Definición de variables de confusión
3. Construcción del DAG causal
4. Estimación del efecto causal
5. Generación de contrafactuales
6. Visualización de resultados
